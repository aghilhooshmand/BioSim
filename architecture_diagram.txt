Data Flow:
CLuB Data (400 features) + cBioPortal Data (4000 features) 
    ↓
Column Pair Definition (Mapping Rules)
    ↓
Preprocessing (Validation, Cleaning, Normalization)
    ↓
BioBERT Model (BioLORD-STAMB2-v1) - Semantic Embedding
    ↓
Similarity Calculation (Cosine Similarity)
    ↓
Result Generation (Filtered, Ranked Matches)

Key Innovation:
Traditional Approach: 400 × 4000 = 1,600,000 brute-force comparisons
BioSim Approach: Semantic similarity reduces to meaningful matches only

Performance Metrics:
- Processing Speed: ~1000 similarity calculations/minute
- Accuracy: Validated against expert annotations
- Scalability: Tested up to 10,000 features
- Similarity Score Range: 0.0 (no similarity) to 1.0 (exact match)

Deployment:
- Docker Containerization
- Streamlit Web Interface (Port 8601)
- GPU Acceleration Support
- Model Caching for Efficiency
